# WebNFC

- 対応端末: Android 5.0 以上
- 対応ブラウザ: Chrome 81 以上
- 対応フォーマット: NDEF
- 対応プロトコル: NFC-A, NFC-B, NFC-F, NFC-V
- 対応モード: Read/Write
- 対応技術: WebNFC API
- 対応言語: JavaScript
- 対応ライブラリ: [webnfc](
- 対応ツール: [Web NFC API Polyfill](
- 対応サービス: [Web NFC API Polyfill Service Worker](
- 対応ドキュメント: [Web NFC API Specification](https://w3c.github.io/web-nfc/)
- 対応リポジトリ: [Web NFC API Polyfill GitHub](
